# heatwave-contract

Published forecast contract for the Thailand sub-seasonal heatwave app.

Served via GitHub Pages: `https://mcteekung.github.io/heatwave-contract/forecast_provinces.json`

See [`CONTRACT.md`](./CONTRACT.md) for the schema. Data is regenerated, validated, and pushed
by the DeepSeek bridge (`publish_bridge.py --publish`); this repo holds only the published
contract, not the model or research code.
